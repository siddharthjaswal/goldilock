# Goldilock Zone

**Goldilock Zone** is a salary‑negotiation simulator that helps people visualize total compensation, tax impact, and negotiation targets in seconds.

> “Not too low, not too high — just right.”

## What it does
- **Offer Builder:** adjust base, bonus, equity, and signing bonus with live sliders
- **Real take‑home:** country + tax profile modeling
- **Side‑by‑side comparison:** current comp vs offer
- **Verdict engine:** plain‑English negotiation guidance
- **Currency toggle:** view in local and target currencies

## Why it matters
Comp packages are confusing. Goldilock Zone makes the tradeoffs visible so users can negotiate with confidence.
